## Название проекта

**Smart Shelf Occupancy Analyzer** — интеллектуальная система для автоматического анализа заполненности торговых полок на основе методов компьютерного зрения и нейросетевых алгоритмов детекции объектов.

### Краткое описание

Программа позволяет автоматически анализировать изображения торговых полок, обнаруживать пустые зоны и вычислять процент заполненности. Система предназначена для использования в розничной торговле с целью автоматизации контроля выкладки товаров, сокращения ручного труда и повышения эффективности мерчандайзинга.

### Основные возможности

- Загрузка и предобработка изображений торговых полок
- Обнаружение пустых зон с использованием нейронной сети Faster R-CNN
- Вычисление процента заполненности полки
- Классификация уровня заполненности (полная/средняя/низкая/критическая)
- Визуализация результатов с ограничивающими рамками
- Сохранение истории анализа в SQLite
